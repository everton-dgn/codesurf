## Overview

**Contex** (`collaborator-clone`) is an Electron 41.x desktop app: an infinite-canvas workspace where tiles (terminal, code editor, browser, kanban, chat) live on a 2D canvas and AI agents interact via MCP. The `feature/hardening-wave-1` branch is the current active branch; prior `feature/event-bus-mcp` is long-merged. Multiple hardening waves landed 2026-06-06/07, resolving all known security issues, substantially decomposing the major god-files, expanding the test suite to 384+ unit tests across 55 files plus 9 E2E specs, and upgrading CI to gate on every PR and push to main.
