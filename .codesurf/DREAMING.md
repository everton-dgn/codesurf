## Overview

**CodeSurf** is an Electron desktop app — an infinite 2D canvas where tiles (terminal, code editor, browser, kanban, chat) live alongside AI agents. Agents connect via a local MCP HTTP server. Humans and agents collaborate asynchronously on shared canvas state.

"Contex" is legacy internal-only (`window.contex`, `~/.contex/`, `contex-relay`). Use "CodeSurf" in all user-facing work.
