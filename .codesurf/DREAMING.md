## Overview

**Contex** (`collaborator-clone`) is an Electron desktop app: an infinite-canvas workspace where tiles (terminal, code editor, browser, kanban, chat) live on a 2D canvas and AI agents interact via MCP. The `feature/event-bus-mcp` branch named in CLAUDE.md is long-merged. HEAD reflects a completed Phase 2/3 hardening plan (all commits 2026-06-06) that decomposed the major god-files, shipped security patches, and brought the test suite to 331 tests across 35 files.
