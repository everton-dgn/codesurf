Recent commits (2026-06-08/09) add GPU transform for interacting tiles, CanvasTileItem extraction with memoization, OWL host runtime with IPC, and global styled title tooltips.

## Codex Provider: Command Surfacing

`src/main/chat/providers/codex.ts` classifies shell commands at ~line 76 (`classifyCodexCommand()`):
- `search`/`read` classified commands (rg/grep/fd/cat/sed/head) → grouped into a single `codex-explore` chip
- `command` classified commands (build, test, publish, dev-server, etc.) → surfaced as individual `exec_command` tool blocks with unique IDs (`codex-cmd-{seq}`), interleaving chronologically with assistant text

Prior to this fix, `command_execution` events were silently dropped unless classified as search/read, causing large blocks of prose with no visible tool calls.

Known Codex model identifiers in use: `gpt-5.5`, `gpt-5.1-codex-max`, `gpt-5`, `gpt-5-codex`
