## Overview

**contex** is a multi-workspace Electron desktop app (codesurf repo) where AI agents and humans collaborate on an infinite 2D canvas. Tiles (terminal, browser, code editor, kanban, chat, layout builder, etc.) live as first-class actors on the canvas. Agents connect via a local HTTP MCP 2.0 server. All state is file-based; no cloud sync.

Active branch: `feature/event-bus-mcp`

## Durable Facts

### Core Architecture
- `src/renderer/src/App.tsx` — ~1700 LOC monolith containing the entire canvas engine (pan/zoom, drag/resize, snapping, groups, undo/redo). Be surgical here; changes ripple everywhere.
- `src/renderer/src/theme.ts` — extracted from App.tsx; ~260 theme tokens. All tile components must use tokens from this file; no inline hardcoded colors except `#1e1e1e`/`#252525`/`#333` globals.
- `src/renderer/src/components/PanelLayout.tsx` — rebuilt from scratch; recursive leaf/split tree, tab strip, per-leaf toolbar. Border model: one 1px `rgba(255,255,255,0.08)` ring on the root container only; no borders on individual pane leaves.
- MCP server starts on a random port; config written to `~/.contex/mcp-server.json`. Never hardcode the port.
- Persistence: `~/.contex/workspaces/{id}/canvas.json` (auto-saved, 500ms debounce), `~/.contex/workspaces/{id}/tiles/{tileId}.json`, `~/.contex/mcp-server.json`.

### Tile System
- All tile types registered in `src/shared/types.ts` `TileType` union.
- Every tile component is lazy-loaded via `React.lazy` + `Suspense`.
- `ThemeBuilderTile` was removed (temporary debug tool, no longer in TileType).
- Active tile types include: terminal, browser, code, kanban, note, image, media, extension, chat, layout-builder.

### Canvas / Group State
- Groups have `layoutMode: 'floating' | 'docked'`. When `'docked'`, the group renders as a `PanelLayout` that fills the window.
- `isPanelMode` is derived from `groups` state in App.tsx. When true, the sidebar and top toolbar are hidden entirely.
- Groups default to `rgba(59,130,246,0.15)` background color (`defaultGroupColor`). `getGroupColor` falls back to this value for groups with no explicit color.
- Undo state holds full canvas snapshots (max 50). Never push to undo stack in hot paths.

### Chat Tile
- Provider selector: Claude (IPC), `claude-direct` (Anthropic SDK bypassing IPC), Codex (CLI subprocess), OpenCode (HTTP server).
- Model chip strip; selected model has `[model-selected]` attribute styled with `workspaceTabActiveBg`.
- Dream completions appear as synthetic tool messages, separate from collapsible regular tool message groups.
- First canvas click selects the tile; subsequent clicks inside `.chat-input-container` pass through to the input.

### Workspace Tab Strip
- Tahoe/Sequoia style: flat bottom, 8px top radius. Active tab: 28px height, `workspaceTabAttachedBottomGap = -2`. Inactive tabs: no background. Close: 8×8px red circle.

### BrowserTile
