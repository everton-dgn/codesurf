## Overview

**CodeSurf** (`collaborator-clone`) is an Electron desktop app: an infinite-canvas workspace where tiles (terminal, code editor, browser, kanban, chat) live on a 2D canvas and AI agents interact via MCP. Active branch is `feature/event-bus-mcp` (untracked file: `src/main/ipc/chat.ts`). Recent merged work includes the extension broker sandbox tier, bundled extension fixes, Codex command surfacing fix, and an archive-sessions race condition fix.

## Recent Fixes (landed 2026-06-10)

### Fable 5 model added

### Archive sessions race condition fixed

- Root cause: `mutateArchivedSessionIds` helper did not exist; bulk "Archive chats" IPC handler was doing an unsafe read-modify-write
- Fix: serialized mutation helper added; wired into the IPC path in `canvas.ts`
- Related code: `src/main/ipc/canvas.ts` (IPC handler) + wherever `archivedSessionIds` is persisted

### Codex `command_execution` filter removed

- Prior behaviour: `classifyCodexCommand()` in `src/main/chat/providers/codex.ts` (~line 76) only surfaced `search`/`read` events; all `command_execution` events were silently dropped, producing walls of prose with no tool calls
- Fix: all command events now surface as `exec_command` tool blocks (`codex-cmd-{seq}`), interleaved with text; `search`/`read` grouped into a single `codex-explore` chip

### Extension broker (landed before current session)

- Power-tier extensions now run in Electron `utilityProcess` sandbox via `src/main/extensions/broker/`
- `JsonRpcPeer` is shared with OWL runtime
- Feature flag + per-manifest `broker` override controls routing; which extensions are brokered by default is not yet confirmed
